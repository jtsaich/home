# Jack Tsai — Personal Site

Personal portfolio site for Jack Tsai (蔡承勳), a Product Engineer currently at GAZAI.

## Tech Stack

- **Framework:** Next.js 15 (App Router, Turbopack)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **i18n:** next-intl — English, 繁體中文, 简体中文

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm start
```
